# collapsible.tsx Analysis

**Path**: `src/components/ui/collapsible.tsx`

## Overview
`Collapsible` allows users to expand/collapse a panel. Built on `@radix-ui/react-collapsible`.

## Detailed Code Analysis

### Components
- Exports primitives directly: `Collapsible`, `CollapsibleTrigger`, `CollapsibleContent`.
- Useful for composing custom collapsible sections without the specific styling of an Accordion (which implies a list of items).
